# POA Retreat Website

POA Retreat is a modern, single-page resort marketing website built with **Next.js** and **React**. The site presents a luxury retreat experience with rich visuals, smooth section-to-section navigation, and an AI-powered concierge assistant for quick guest questions.

## What the website includes

- **Hero + resort overview** to introduce POA Retreat’s atmosphere and value.
- **Accommodation and dining section** describing room styles and culinary experiences.
- **Activities and facilities showcase** covering wellness, recreation, and amenities.
- **Photo gallery** to highlight the property and guest experience visually.
- **Booking inquiry form** for availability and special request outreach.
- **AI Guest Concierge** powered by Genkit + Google AI to answer common guest questions.

## Tech stack

- **Framework:** Next.js 15 (App Router)
- **UI:** React 19 + Tailwind CSS + Radix UI components
- **AI:** Genkit with Google GenAI integration
- **Language:** TypeScript

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open `http://localhost:9002`.

## Useful scripts

- `npm run dev` – start local dev server on port 9002.
- `npm run build` – production build.
- `npm run start` – run production server.
- `npm run lint` – run Next.js lint checks.
- `npm run typecheck` – run TypeScript type checks.
- `npm run genkit:dev` – start Genkit development runtime.
- `npm run genkit:watch` – start Genkit in watch mode.

## Project goal

This project is designed to provide an elegant, conversion-friendly website experience for a boutique resort brand, combining strong storytelling, visual presentation, and AI-assisted guest support.
